= NAME
tag   - define tags to this page

= SYNOPSIS
\\{tag tag1 ...}

= DESCRIPTION

Juli tag macro defines tags to this page.

== Example

This:

  \{tag DIY BOOK}

defines 'DIY' and 'BOOK' tags to this page for tag-search later.
